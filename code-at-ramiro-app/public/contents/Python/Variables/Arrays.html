Lists (Arrays)

[Ordenemos, información!]

Como podreis comprobar a medida que os adentreis en el mundo de programación, los objetos (integers, floats, strings, etc.) que componen un lenguaje de programación son empleados como queramos para realizar todo tipo de tareas. Con esto en mente, introducimos las listas. En esencia las listas son contenedores para secuencias de integers, floats, cadenas de texto, etc. Tienen la propiedad ser iterables e indexables. Es decir, son secuencias ordenadas por las que podemas iterar ("pasar por ellas"). Ilustremos con algunas listas(arrays):


    array = [1, "two", "3", 4, 4.5]
    '''Es casi idéntico definir una lista mediante:'''
    array = list(1, "two", "3", 4, 4.5)

Ahora veamos la utilidad de esto. En las listas podemos acceder a elementos de la lista mediante su índice: 


    print(array[0], array[2], array[5]) #tambíen se puede escribir el print() de este modo, e imprimir los argumentos en una misma línea

¿Por qué da error al pedir el 5 elemento de la lista? la respuesta es que en los lenguajes de programación se empieza a "contar" por 0, en vez de 1.\n Veamos ahora lo que es el "slicing", el acceder a numerosos elementos de una lista.

Esto es un párrafo con un link en el 


  print(array[2:4])
  print(array[::])
  print(array[1::2])
  print(array[::-1])

Como se pueden acceder a los 3 hasta el 4 (incluido), a todos los elementos, los elementos a partir del 2 y cogiendo cada 2º elemento, coger todos los elementos pero en orden contrario, etc. De un manera más general, el array[a:b:c] empieza desde el a-1, acaba en el b-1 y coge cada cº elemento. Es decir, podemos pensar esto como un intervalo cerrado que coge de la secuencia cada c veces. El default es coger todo en orden (array[::]).

Algo que vermos más adelante, es que las listas son empleados muchas veces con los for loop tanto para generarlas como para utilizarla la información que contienen. Pondré un ejemplo (que se explicará muy brevemente):


  for i in array:
      print(i)

  for i in [array ::-1]:
    print(i)

Como veis, estos for loops están iterando por la lista. Es decir, primero cogen el 0º elemento, luego el 1º y así sucesivamente hasta que no hay otro elemento en la lista "array".

Al igual que con el resto de objetos, las utilidades son infinitas.En particular, las listas nos permiten guardar información que luego podemos acceder y emplear como se quiera.\n Para cerrar esta subscción, adjuntaré un link a la que podreis acceder os métodos y más funcionalidades que tienen las listas.
