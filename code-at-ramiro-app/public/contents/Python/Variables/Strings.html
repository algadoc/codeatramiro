Strings (Cadena de Texto)

Cadenas de texto

Al igual que se pueden guardar números (integers y floats), es posible almacenar strings(cadenas de texto) en una variable. Sin embargo, a diferencia de los integers y floats es necesario establecer que la cadena de texto a la que estamos asignando no es una variable y para ello se emplean las ' ' o " " Me explico: 


  a = hola

  print(a)

  '''Prints:
  Traceback (most recent call last):
  File "", line 0, in 
    a = hola
NameError: name 'hola' is not defined'''

Al no haber contenido hola en ' ' como 'hola', la shell de python nos devuelve un NameError, ya que interpreta hola como una variable más, que en este caso no está definida. Para evitar este error es necesario contener la cadena de text que deseamos en unas ' ', como sigue en el ejemplo.


a = 'hola'

print(a)

#prints: hola

Sabiendo como guardar una string en una variable, se puede hacer una infinidad de cosas con ellas y aprovechar muchas propiedades que pueden resultar muy útiles. Para ilustrar lo que digo, ejemplificaré algunas operaciones y métodos (que ya conoceremos con más detalle más adelante) que se pueden emplear con las string. Pero como digo, hay una gran cantidad de cosas que se pueden hacer con las strings y para ello, os dejaremos aquí una cheat sheet a la que podreis acceder siempre que querais.


    '''concatenation'''
b = 'Hello ' + 'World'

print(b)

#prints: Hello World

c = 5*'Hello World'

print(c)
#prints: Hello WorldHello WorldHello WorldHello WorldHello World

Como también veremos con los lists en la siguiente subsección, podemos slicear los strings, es decir escoger una porción de la string original, para ello se hace string[start:end] . Es importante notar que es un intervalo no inclusivo, no incluye la posición de end sino que coge hasta end - 1.

Además es útil conocer que el default de start es 0 y el de end es hasta todo el string. Clarifiquemos


    '''string slicing'''
b = 'HeAlexllo'
     012345678  

c = b[2:6] # string[start:end] -> Python starts with index 0!!!!

print(c)
#prints: Alex

d = b[:2] + b[6:] + ' ' + c # b[0:2] = b[:2] y b[6:] = b[6:9]

print(d)
#prints: Hello Alex

Ejemplos de métodos. Aquí adelanto con muy poco detalle la existencia de métodos y funciones, pero la intención es mostrar algunas cosas que se pueden hacer con stringsy la interrelación entre los distintos objetos


  '''splitting una string'''
a = 'DFdjvnfnv-kmfkfinirn'

b = a.split('-') # separa la string por el carácter '-', el default es el espacio ' '

print(b)
#prints: ['DFdjvnfnv', kmfkfinirn ]

d = '1Hello2Hello3Hello4'

e = d.split('Hello')

print(e)
#prints: [1, 2, 3, 4]


        
 

'''replace'''

b = a.replace('a', 'i')
