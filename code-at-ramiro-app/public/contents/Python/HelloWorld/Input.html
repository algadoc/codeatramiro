Input()

O como darle información a la máquina

Input es el comando que nos permite darle información a la máquina dinámicamente, es decir, sin tener que escribirla directamente en el programa. Esto es primordial si queremos hacer un programa que otras personas, en especial no técnicas, puedan usar con facilidad.

Para guardar esta información necesitaremos una variable. Entraremos más en detalle en el próximo apartado, pero por ahora solo necesitas saber que una variable guarda una información. Para usar input, lo igualaremos a una variable de esta manera:

a = input()

Este comando hará que el programa pare de ejecutarse hasta que el usuario introduzca información dentro de la linea de comando. Esa información se guardará en la variable a.

Estudiaremos las variables mas adelate. Para hacer la experiencia del usuario mas agradable, input permite que le demos un texto que aparecerá antes de el input, para que el usuario sepa que tiene que aportar. Esto hacer que las próximas dos piezas de código sean equivalentes

print("Buenas! Por favor, introduce tu nombre")
a = input()
a = input("Buenas! Por favor, introduce tu nombre")
