O como ver lo que la máquina piensa

Ahora que sabemos como interactuar con la máquina virtual de Python, vamos a aprender nuestros primeros comandos. Estos comandos son las instrucciones que podemos dar a nuestra máquina. Para empezar, vamos a aprender como hacer que la máquina devuelva algo, es decir, que escriba a la consola un mensaje. Para ello, vamos a usar el comando print()

Podemos apreciar también parte de la sintaxis propia de Python. Para usar un comando, tenderemos primero el nombre del comando (en este caso print) y luego unos paréntesis donde daremos la información que necesita el comando para funcionar.

Para escribir algo en la consola haremos lo siguiente: escribiremos el comando print, y dentro de sus paréntesis escribiremos el mensaje que queremos que se escriba.

print("Hello world!")

Si te fijas en el comando, necesitamos poner el mensaje entre comillas. No necesitamos esto si queremos que escriba un número

print(64920)

Entenderás por qué esta diferencia en el proximo apartado de variables (tiene que ver con el tipo de información y como guarda el ordenador la información).
