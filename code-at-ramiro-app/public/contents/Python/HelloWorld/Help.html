Help()

O como encontrar qué es un comando en un apuro

Help() es un comando fantástico si no sabes exactamente que hace un comando o que es una variable. Con el, puedes ver qué es qué sin tener que buscarlo en internet (aunque siempre es muy recomendable).

Para usuarlo, meteremos entre los paréntesis de lo que queremos saber. Por ejemplo, para saber lo que hace el comando print() podemos ejecutar el siguiente código:

help(print)

Esto nos devolverá el siguiente parrafazo:

Help on built-in function print in module builtins:

print(...)
    print(value, ..., sep=' ', end='\n', file=sys.stdout, flush=False)
    
    Prints the values to a stream, or to sys.stdout by default.
    Optional keyword arguments:
    file:  a file-like object (stream); defaults to the current sys.stdout.
    sep:   string inserted between values, default a space.
    end:   string appended after the last value, default a newline.
    flush: whether to forcibly flush the stream.
(END)

Conforme avancemos, iremos entendiendo qué es cada una de las cosas de las que el comando help habla. Por ahora, con conocerlo basta.
