La máquina virtual de Python

Donde ocurre la magia

Ahora que ya sabemos un poco sobre qué es Python y como funciona, vamos a empezar a utilizar esa máquina virtual de la que hablábamos en el apartado anterior. Para hacerlo, usaremos un programa llamado consola. No, no es una Playstation. Una consola (o CLI, command-line-interface) es un programa que acepta una entrada de texto para recibir información e instrucciones. Tu ordenador tiene una, se llama terminal. Todo lo que haces visualmente en la pantalla del ordenador en realidad se está tranformando en comandos que son procesados por el ordenador como instrucciones a través de una terminal. Con Python, es exactamente igual.

Una terminal, en este caso ejecutando un programa llamado npm

Para empezar a programar, tenemos primero que instalar Python. Para ello, tienes que irte a python.org/downloads/ y seguir las instrucciones detalladas en la página para tu sistema operativo (Windows o Mac).

Una vez hayas instalado Python, podrás utilizar un programa llamado IDLE. En este programa, podrás empezar a escribir scripts y interactuar con la mñaquina virtual.

En la sesión del próximo lunes empezaremos a programar con la consola, y aprenderemos a hacer nuestro primer script. ¡Que emoción!
